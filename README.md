# Tarantula Adventure Website

Website resmi organisasi Tarantula Adventure – wadah penggiat alam mahasiswa Universitas BSI Yogyakarta.

## Deskripsi

Website ini dibuat sebagai media informasi dan dokumentasi kegiatan Tarantula Adventure.  
Berisi halaman:

- Home
- Tentang
- Kegiatan
- Berita & Artikel
- Galeri
- Kontak (terintegrasi WhatsApp)

Website dibuat menggunakan HTML, CSS, dan JavaScript dengan desain responsif (mobile-friendly).

---

## Fitur Utama

- Navbar responsif dengan hamburger menu
- Grid layout modern untuk Kegiatan & Artikel
- Galeri dengan fitur lightbox (klik untuk perbesar)
- Form kontak langsung terhubung ke WhatsApp
- Desain clean dan profesional
- Fully responsive (Desktop, Tablet, Mobile)

---

## Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
